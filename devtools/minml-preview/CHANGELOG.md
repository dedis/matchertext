# Change Log

All notable changes to the "minml-preview" extension will be documented in this file.

## [0.0.1] - 2026-03-12

### Added
- Initial implementation of the MinML Live Preview extension.
- Integration with Go-based WASM parser for accurate rendering.
- Support for local images using the `<base>` tag in the webview.
- Real-time updates as you type in the editor.
- Cross-platform installation via the root `Makefile`.
